  • 初始化服务端

最后输出 Game Server started on port 22102 的字样代表启动成功

在终端输入stop来停止运行

  • 修改配置文件

键入i切换INSERT模式，把文件中的两个IP地址127.0.0.1修改为你服务器的地址，修改完成后按 esc 输入 :wq 保存

  • 运行服务端

最后停止在 Game Server started on port 22102 的字样

在终端输入 account create [username] {playerid} 来创建账号

更多终端指令请查看 Grasscutters/Grasscutter: A server software reimplementation for a certain anime game. (github.com)

  • 获取证书

目录下的文件说明如下：

  • mitmproxy PEM格式的证书和私钥。
  • mitmproxy-ca-cert.pem PEM格式的证书。使用它可以在大多数非Windows平台上分发。
  • mitmproxy-ca-cert.p12 PKCS12格式的证书。用于Windows。
  • mitmproxy-ca-cert.cer 与.pem相同的文件，但某些Android设备需要扩展名。

这里我们将 mitmproxy-ca-cert.p12 下载到自己的电脑上
